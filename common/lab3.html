Лабораторная работа №3

Ветвления

Изучить разветвляющиеся алгоритмы, операторы выбора, программирование разветвляющегося вычислительного процесса.

Задача 1

Даны два целых числа: A, B. Проверить истинность высказывания: «Справедливы неравенства A ≥ 0 или B < −2».

Задача 2

Даны три целых числа. Найти количество положительных и количество отрицательных чисел в исходном наборе.

Задача 3

Вычислить значение функции y = f(x) при произвольных значениях x. y - кусочная функция:

  • при x ≤ -2, y = 2 * (cos(a * x² - b))²;
  • при x > 3, y = 3 * x² + b;
  • при -2 < x ≤ 3, y = (x² + e^(a * x))¹/².

Где a = 1.3, b = 2.5.

Задача 4

Вычислить значение функции y = f(x) для трех значений x: -2, 4, 3.
